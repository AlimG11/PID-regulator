# PID-regulator

## Параметры командной строки

Программа поддерживает настройку параметров моделирования через аргументы командной строки.

### Общий формат запуска

```bash
python pendulum_2link_pid.py [параметры]
```

### Доступные параметры

| Параметр | Описание | Значение по умолчанию |
|-----------|-----------|-----------|
| `--gui` | Запуск симуляции в графическом интерфейсе PyBullet | выключен |
| `--tmax` | Время моделирования в секундах | `15.0` |
| `--qdes Q1 Q2` | Желаемые углы первого и второго сустава (рад) | `0.8 -0.6` |
| `--q0 Q1 Q2` | Начальные углы первого и второго сустава (рад) | `0.0 0.0` |
| `--urdf FILE` | Путь к URDF-файлу модели | `two_link_pendulum.urdf` |
| `--outfile FILE` | Имя файла для сохранения графиков | `two_link_pid_result.png` |

### Примеры запуска

Запуск с параметрами по умолчанию:

```bash
python pendulum_2link_pid.py
```

Запуск с отображением окна PyBullet:

```bash
python pendulum_2link_pid.py --gui
```

Увеличение времени моделирования до 25 секунд:

```bash
python pendulum_2link_pid.py --tmax 25
```

Задание другого целевого положения суставов:

```bash
python pendulum_2link_pid.py --qdes 1.0 -0.5
```

Задание начального положения суставов:

```bash
python pendulum_2link_pid.py --q0 0.3 -0.2
```

Указание собственного URDF-файла:

```bash
python pendulum_2link_pid.py --urdf my_robot.urdf
```

Сохранение графиков под другим именем:

```bash
python pendulum_2link_pid.py --outfile result.png
```

Полный пример:

```bash
python pendulum_2link_pid.py --gui --tmax 25 --qdes 1.0 -0.5 --q0 0.2 0.0 --urdf two_link_pendulum.urdf --outfile experiment.png
```

### Результаты работы программы

После завершения моделирования программа:

1. Выводит в консоль итоговые углы суставов.
2. Выводит ошибку позиционирования в радианах.
3. Строит графики:
   - углов суставов;
   - управляющих моментов;
   - траектории конечного эффектора;
   - абсолютной ошибки управления.
4. Сохраняет графики в файл, указанный параметром `--outfile`.
